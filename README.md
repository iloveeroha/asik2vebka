# Weather & Country News App

## Description
This project is a simple backend + frontend web application built with Node.js and Express.
It allows users to:
- Get weather information for any city in the world using the OpenWeatherMap API.
- Get latest news related to any country using the NewsAPI.

The project demonstrates working with external APIs, backend routing, and frontend interaction using fetch.

---

## Technologies Used
- Node.js
- Express.js
- Axios
- HTML, CSS, JavaScript
- OpenWeatherMap API
- NewsAPI

---

## Features

### Weather
- User can enter any city name.
- Weather data is fetched from OpenWeatherMap API.
- Displays temperature, weather description, wind speed, rain, country code and coordinates.

### Country News
- User can enter any country name.
- News related to the entered country are fetched using NewsAPI.
- Displays latest news headlines with links.
